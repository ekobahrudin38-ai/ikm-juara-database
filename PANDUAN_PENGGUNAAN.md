### 1. Dashboard
- Melihat statistik total IKM, HKI, Sertifikat Halal, dan Pelatihan
- Melihat aktivitas terbaru dari admin dan pengguna

### 2. IKM Binaan
**Menambah Data IKM:**
1. Klik tombol **"+ Tambah IKM Binaan"**
2. Isi form:
   - No. NIB (13 digit)
   - No. NIK (16 digit)
   - Nama Lengkap
   - Alamat Lengkap
   - Nama Usaha
   - Nomor HP
3. Klik **"Simpan Data"**

**Fitur Lain:**
- Export ke Excel/PDF
- Edit data
- Hapus data (masuk Recycle Bin)

### 3. Layanan IKM JUARA

#### A. Pendaftaran HKI Merek
1. Klik **"+ Tambah Data HKI"**
2. Cari data IKM dengan NIB/NIK/Nama
3. Klik **"Gunakan Data"** (data IKM akan otomatis terisi)
4. Isi data tambahan:
   - Nomor Pendaftaran HKI
   - Link Bukti Daftar HKI (Google Drive)
   - Link Surat Pernyataan (Google Drive)
   - Status Sertifikat (Telah Didaftar/Proses/Ditolak)
   - Tahun Fasilitasi
   - Link Sertifikat HKI (opsional)
5. Klik **"Simpan Data"**

#### B. Pendaftaran Sertifikat Halal
1. Klik **"+ Tambah Sertifikat Halal"**
2. Cari dan gunakan data IKM
3. Isi data tambahan:
   - Nomor Sertifikat Halal
   - Link Sertifikat Halal (Google Drive)
   - Link Logo Sertifikat (Google Drive)
   - Tahun Fasilitasi
   - Status Sertifikat
4. Klik **"Simpan Data"**

#### C. Pendaftaran TKDN IK
1. Klik **"+ Tambah TKDN IK"**
2. Cari dan gunakan data IKM
3. Isi data tambahan:
   - Nomor Sertifikat TKDN IK
   - Link Sertifikat (Google Drive)
   - Tahun Terbit Sertifikat
   - Status Sertifikat
4. Klik **"Simpan Data"**

#### D. Pendaftaran SIINas
1. Klik **"+ Tambah SIINas"**
2. Cari dan gunakan data IKM
3. Isi data tambahan:
   - Nomor Bukti Kepemilikan Akun
   - Tahun Bukti Kepemilikan
   - Link Bukti (Google Drive)
4. Klik **"Simpan Data"**

#### E. Pendaftaran Uji Nilai Gizi
1. Klik **"+ Tambah Uji Nilai Gizi"**
2. Cari dan gunakan data IKM
3. Isi data tambahan:
   - Nomor Laporan Hasil Uji
   - Tanggal Hasil Uji
   - Tahun Fasilitasi
   - Link Laporan (Google Drive)
4. Klik **"Simpan Data"**

#### F. Pendaftaran Kurasi Produk
1. Klik **"+ Tambah Kurasi Produk"**
2. Cari dan gunakan data IKM
3. Isi data tambahan:
   - Nomor Sertifikat Kurasi
   - Link Sertifikat (Google Drive)
4. Klik **"Simpan Data"**

### 4. Pelatihan Pemberdayaan Industri

**Menambah Pelatihan:**
1. Klik **"+ Tambah Pelatihan"**
2. Isi form:
   - Jenis Pelatihan
   - Sub Kegiatan
   - Tahun Pelaksanaan
3. Klik **"Simpan Data"**

**Menambah Peserta Pelatihan:**
1. Klik **"Lihat Peserta"** pada pelatihan yang dipilih
2. Klik **"+ Tambah Peserta"**
3. Cari data IKM dengan NIB/NIK/Nama
4. Klik **"Gunakan Data"**
5. Pilih jenis pelatihan (bisa lebih dari 1)
6. Klik **"Simpan Data"**

**Catatan:** Jumlah peserta akan otomatis terupdate

### 5. Penelusuran Data
1. Masukkan NIB (13 digit), NIK (16 digit), atau Nama Lengkap
2. Klik **"Cari Data"**
3. Sistem akan menampilkan:
   - Data IKM Binaan
   - Data HKI Merek
   - Data Sertifikat Halal
   - Data TKDN IK
   - Data SIINas
   - Data Uji Nilai Gizi
   - Data Kurasi Produk
   - Data Pelatihan yang diikuti

### 6. Recycle Bin
- Melihat data yang telah dihapus
- **Restore:** Mengembalikan data ke dashboard semula
- **Hapus Permanen:** Menghapus data secara permanen

### 7. Laporan
1. Pilih filter:
   - Jenis Layanan (Semua/IKM/HKI/Halal/dll)
   - Tahun
2. Klik **"Export Excel"** atau **"Export PDF"**

### 8. Log Aktivitas
- Melihat semua aktivitas admin dan pengguna
- Informasi yang tercatat:
  - Tipe User (Admin/Pengguna)
  - Nama User
  - Aktivitas yang dilakukan
  - Detail aktivitas
  - Waktu aktivitas
- Export ke Excel/PDF

## ğŸ” Tips Pencarian

### Format Input:
- **NIB:** 13 digit angka (contoh: 1234567890123)
- **NIK:** 16 digit angka (contoh: 3578012345670001)
- **Nama:** Bisa sebagian nama (contoh: "Budi" akan menemukan "Budi Santoso")

### Pencarian Case-Insensitive:
- "budi santoso" = "Budi Santoso" = "BUDI SANTOSO"
